⚡ Sequence Diagram
Luồng xử lý từng tin nhắn theo thời gian — từ khi user gửi đến khi nhận response. Hai luồng chính: Order Query (OrderLookupTool) và General Query (Agentic RAG).
Luồng 1
Order Query
Detect mã đơn / SĐT → SQL cố định → reply ngay. Không qua RAG. ~200ms
Luồng 2
General + Emotion
DeepSeek chat → reply. Song song: Orchestrator RAG → emotion. ~2s
Speed Layer
Mọi tin nhắn
Sau mỗi response → ingest vào SpeedLayer. Alert nếu angry/disappointed.
